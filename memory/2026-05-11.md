3. **Patched server.js** — removed fake CPU/MEM injection in agents endpoint
4. **Patched app.js** — removed `triggerOptimization()` fake logic, removed CPU/MEM from agent detail panel, updated empty state messages
5. **Patched index.html** — removed "Trigger Optimization" button from subagents page
6. **Verified** — no fake agent names remain in any source file

### Key lesson:
**Never seed fake data. Not for demos, not for "placeholders", not for "coming soon". If it's not real, it doesn't go in.**
