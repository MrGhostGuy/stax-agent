# 2026-05-16

## Claw3D Chat Fix + Performance Optimization (21:33-22:30)

**Problem:** Claw3D agents couldn't respond in chat, and the 3D office was very laggy.

### Root Causes Found:

1. **Chat not responding:**
   - `chat.send` via gateway had no timeout — if the gateway was slow/stalled, the call would hang indefinitely
   - When streaming responses never completed (e.g., gateway disconnect), agents stayed in "running" state forever with no feedback
   - The `handleChatSend` in OfficeScreen.tsx had a logic gap where office trigger intents (desk, gym, etc.) were processed but the message was still sent to the agent — this was actually correct behavior but the agent would appear to not respond because the streaming timeout was too long

2. **Lag:**
   - Canvas rendering at full frame rate (60fps) with expensive A* pathfinding every frame
   - DPR (device pixel ratio) was too high (up to 1.5) and adjusted every 45 frames
   - `officeAgents` memo recalculated every 2 seconds due to `clockTick` dependency
   - `loadAgents` polling every 60s with expensive session inference
   - No path caching — A* recomputed for identical start/end positions every frame
   - `RetroOffice3D` not memoized — re-rendered on every parent state change
   - Environment map loaded as background (expensive HDR)
   - Antialiasing enabled (expensive for 2D orthographic scene)

### Fixes Applied:

**`src/features/agents/operations/chatSendOperation.ts`:**
- Added 15-second timeout on `chat.send` via `Promise.race`
- Added 30-second safety timeout for streaming responses — auto-resets agent to "idle" if stream never completes
- Added module-level `streamTimeouts` registry with `registerStreamTimeout` and `clearStreamTimeout` exports
- Added "(no response)" message when terminal-immediate response has no text

**`src/features/retro-office/RetroOffice3D.tsx`:**
- Added A* path cache (`pathCacheRef`) — caches paths for 5 seconds, prunes at 200 entries
- Reduced DPR range from `[0.85, 1.5]` to `[0.75, 1.25]`
- Changed shadow type from `PCFShadowMap` to `PCFSoftShadowMap` (faster)
- Disabled antialiasing (`antialias: false`)
- Disabled stencil buffer
- Set `background={false}` on Environment (no HDR background render)
- Wrapped component in `memo()` to prevent unnecessary re-renders
- Increased DPR adjustment interval from 45 frames to 60 frames (2 seconds at 30fps)

**`src/features/office/screens/OfficeScreen.tsx`:**
- Increased `clockTick` interval from 2s to 5s (reduces `officeAgents` recalculation)
- Increased `loadAgents` polling interval from 60s to 120s
- Increased `minIntervalMs` from 60s to 120s, `onlyWhenIdleForMs` from 120s to 180s
- Removed `void clockTick` dependency from `officeAgents` memo (was forcing recalc)
